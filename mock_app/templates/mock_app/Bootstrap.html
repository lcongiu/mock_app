Integration with Bootstrap

If we inspect the html of this or any other page, we will see that the code links to a boostrap css file (see the file /templates/mock_app/Bootstrap.html for details). This means that oTree is natively integrated with Bootstrap 4.

Specifically, it implies that oTree:

  1. is natively responsive, i.e. its elements scale with screen width
  2. supports built-in Bootstrap classes ("container", "btn-primary", etc.)

For example, if you try to resize this page, you will see that this content shrinks to fit the screen width.

If you inspect the code of this page, you will see that the content of this page has been encapsulated into a div of class "otree-body container". This occurs by default, and serves as the html body. The class "container" is one of the main classes of bootstrap, and is responsive by default.

Again, you should refer to the customization of this class in the file style.css. But you can also customize it in the html code of the page of interest, as will be shown in a second.
