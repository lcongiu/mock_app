Structure

Each html page extends a base template (named "Page.html"), which contains the base html structure of a page. In fact, in each page you will find just the Django block that extends "global/Page.html", without containing html code per se. This is because the html code needed to structure a page is imported from Page.html.

If you inspect the present or any other oTree page on the browser (right-click -> inspect), you will see that some html code has been rendered without it being present directly in the corresponding html file. For example, the document begins with the tags "!DOCTYPE html", "html", "head", required to define a html document, even though the html file in the App does not contain this code (you can check it in the file templates/mock_app/Structure.html).

An implication of it is that you can customize the "source" page, i.e. Page.html, with the features you would like each html page to have. Page.html can be found inside the files of the App, at the path: "_templates/global/Page.html"

In this template App, Page.html has been modified so to include a basic .css file to style the main elements of each page, such as the position of titles and "next" button. For details, look at "style.css" in "_static/global". Of course, these settings can be overwritten in each html page, if needed.
